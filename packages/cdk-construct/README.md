# Overview

This is the Release Console for the MicroApps framework.

# Development

- `nvm use`
- For Mac
  - Install Xcode
  - `xcode-select --install`
  - `brew install vips`
- `npm i`
- `npm run dev`
- Open in browser: `http://localhost:3000/nextjs-demo/0.0.0`
